# blog.adarshd.dev

Personal tech blog by Adarsh Divakaran — mostly Python, some AI, occasional personal posts.

Built with [Eleventy](https://www.11ty.dev/) and deployed via GitHub Pages.

## Local Development

```bash
npm install
npm run serve
```

Site will be available at `http://localhost:8080`.

## Build

```bash
npm run build
```

Output goes to `_site/`.
